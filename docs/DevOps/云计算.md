# 云计算

云计算是一种按量付费的模式，可以提供可用的、便捷的、按需的网络访问与资源共享。云计算大幅促成了 $\mathrm{DevOps}$ 的发展。

服务模式：自建、$\mathrm{SaaS}$、$\mathrm{IaaS}$、$\mathrm{PaaS}$

部署模式：私有云、社区云、共有云等

## 边缘计算

在网络边缘（即，**靠近数据源头的一段**）处理大量临时数据，具有以下特点：

- 计算任务不断分解和下发；
- 具备低延时、高可靠性以及数据安全的要求；
- 与云计算互补，解决了云计算的某些短板。

## 边缘计算

目前，边缘计算还面临着以下的挑战：

- 可编程性：$\mathrm{IoT}$ 设备的不断涌现，需要边缘计算面对各种异构平台和系统
- 数据抽象：数据具有多样性，其频率和格式差异较大。
- 服务管理：利用 $\mathrm{DEIR}$ 模型来进行高效管理。
- 数据隐私保护和安全：隐私、数据权属等。
- 理论基础：多学科协同交叉。
- 商业模式：协调多方利益相关者。

## 雾计算

主要用来填补云（云计算）和物（边缘计算）之间的计算需求。雾运算使用最终用户终端设备或连接最终用户设备的边缘设备，以分散式协作架构进行资料存储（相较于将资料集中存储在云端数据中心），或进行分散式网路封包传输通信（相较于透过互联网骨干路由），或相关分散式控制或管理。
