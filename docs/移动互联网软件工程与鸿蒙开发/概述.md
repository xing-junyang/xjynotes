# 移动互联网软件工程概述

## 主要内容

- 道（**软件工程方法学、数据管理**）
- 法（**云原生、数据治理**）
- 术（**客户端服务器开发、基础设施建设**）
- 器（**工具链**）

**软件工程方法学**：$\mathrm{SWEBOK}$

**数据管理**：一项复杂的学科，包括数据储存和操作、大数据科学、数据建模设计等。
