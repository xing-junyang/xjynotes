# 人机交互系统概述

## 什么是人机交互

**人机交互** $\text{(HCI, CHI, or Human–Computer Interaction)}$，是研究**系统与用户之间的交互关系的学问**。在八十年代被独立出来研究，目标在于：解决终端用户和技术的“**最后一公里**”。人机交互的研究提出了很多如交互设计、用户测试、用户中心设计 $\text{(UCD)}$、用户体验等概念和思想，不仅在现实中也得到了广泛的应用，还对未来新技术的如何呈现提出了新的思考。

::: tip 人机交互、用户体验和交互设计的辨析

**人机交互**是较为学术化的词汇，常见于计算机专业研究人员，以开发的角度强调了系统与用户之间的交互关系，如今也包含体验的诸多方面。

**用户体验**是一个更为广泛的概念，拥有广泛的用户作为基础，常用于产品设计、工业设计等商业领域。通常围绕终端用户与公司产品与服务交互的各个方面展开。不能直接设计用户体验，而是围绕用户体验来设计。

**交互设计**是一个更为具体的概念，泛指关于交互式产品和服务的设计。

:::

## 日常生活中的交互设计（的缺陷）

- 银行的自动应答系统；
- 电子商务网站上，中国（China）并不在 $C$ 开头的国家列表；
- 返回桌面图标的位置，$\mathrm{macOS}$ 和 $\mathrm{Windows}$ 平台的菜单栏位置。

### 为什么会出现上述问题

- 设计人员假设对于技术的使用方式的理解可以通过他们的自主思考实现，即想象这个技术是如何被使用的。
- 设计人员假设每个人都是相同的。并未采取**用户中心**的设计方法。

## $\mathrm{HCI}$ 的重要性

- **市场角度**
    - 用户期望简单易用的系统
    - 对设计低劣的系统容忍度越来越差
- **企业角度**
    - 提高员工的生产效率
    - 降低产品的开发成本
    - **降低产品的后续支持成本**
- **用户角度**
    - 获得较高的满意度
    - 减少时间、金钱和生命损失

## 结语

人机交互是软件工程人员需要掌握的核心知识领域之一，同时也是最易被忽略的学科。此外，人机交互也较难被完全掌握，它是典型的**交叉学科**，与**心理学、人类学、设计学**等学科有着密切的联系。
