# 简介

::: tip 初次使用

您可以在**左侧的导航栏**上查看本网站的所有内容，也可以使用**左上角的搜索框**进行搜索。页面**右侧的目录栏**显示了当前页面的大纲。

对于移动用户，**导航栏**位于上方横条的左侧 `Menu` 菜单中，**目录栏**位于上方横条的右侧 `本页面` 下拉菜单中，可以点击最上方的**按钮**进行**搜索**。

:::

现今，网站的内容主要包括以下几个方面：

- **软件工程**领域相关：**软件工程基础、软件测试、人机交互**、移动互联网软工、开发运维一体化等。
- 传统**计算机科学**：**自动机理论**、**`C++` 编程语言**、操作系统、编译原理等。
- **机器学习**；
- **杂项**：鸿蒙开发、嵌入式、虚拟机配置、数海札记（一个分享数学上的有趣问题的专栏）、照片集等。

后续也会继续更新和完善现有内容，或新增其他类别。
